# Recruitment Analytics Dashboard

## Overview
The Recruitment Analytics Dashboard is an Excel-based project developed to analyze and track recruitment-related metrics through interactive reports and visualizations. The dashboard helps in monitoring candidate pipeline, hiring status, department-wise hiring, and recruitment performance in a structured manner.

## Features
- Candidate pipeline tracking
- Offer status analysis
- Department-wise hiring insights
- Recruitment performance visualization
- KPI-based recruitment reporting

## Dashboard Metrics
- Total Candidates
- Selected Candidates
- Rejected Candidates
- Pending Candidates

## Visualizations Included
- Offer Status Pie Chart
- Department-wise Hiring Bar Chart

## Tools Used
- Microsoft Excel
- Excel Charts & Visualizations
- Excel Formulas

## Purpose of the Project
This project demonstrates reporting, dashboard preparation, and analytical skills relevant to recruitment and Talent Acquisition processes.

## Project Outcome
The dashboard provides a clear and organized view of recruitment data, helping in better tracking, reporting, and recruitment analysis.
